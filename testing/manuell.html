Manuelle Tests

Schnelltest-Checkliste

Diese Checks können bei jeder Entwicklung schnell durchgeführt werden:

  1. Tab-Test: Alle interaktiven Elemente mit Tab erreichbar? Fokus sichtbar? Logische Reihenfolge?
  2. Zoom-Test: Seite auf 200% und 400% zoomen. Kein Inhaltsverlust? Kein horizontales Scrollen bei 320px?
  3. Heading-Test: Überschriften-Hierarchie prüfen (Browser-Extension oder DevTools).
  4. Kontrast-Test: Texte und UI-Elemente mit Contrast Checker prüfen.
  5. Nur-Tastatur-Test: Maus weglegen. Kann alles bedient werden?
  6. Bild-Test: Bilder deaktivieren. Geht Information verloren?
  7. Motion-Test: prefers-reduced-motion aktivieren. Animationen reduziert?

Ausführlicher Test

Für umfassende Prüfungen (z.B. vor einem Release) zusätzlich:

  1. Screenreader-Test (siehe Screenreader-Tests)
  2. Test mit Sprachsteuerung (Voice Control / Dragon)
  3. Test mit Vergrösserungssoftware (ZoomText, Browser-Zoom)
  4. Test mit aktiviertem High-Contrast-Mode
  5. Prüfung aller Formulare inkl. Fehlermeldungen
  6. Prüfung aller dynamischen Inhalte (Dialoge, Live-Updates, Lazy Loading)

Text-Spacing-Test Muss

Folgende CSS-Werte im Browser überschreiben und prüfen, dass kein Inhalt abgeschnitten wird oder verloren geht:

* {
  line-height: 1.5 !important;
  letter-spacing: 0.12em !important;
  word-spacing: 0.16em !important;
}
p {
  margin-bottom: 2em !important;
}
